# Scripts Utilitaires

Ce dossier contient les scripts utilitaires pour le projet NexTraction.

## Scripts disponibles

### `cleanup_old_tests.py`
Script pour supprimer les anciens fichiers de test du répertoire racine.
```bash
python scripts/cleanup_old_tests.py
```

### `evaluation.py`
Script d'évaluation du pipeline RAG avec des exemples de questions.
```bash
python scripts/evaluation.py
```

## Utilisation

Tous les scripts peuvent être exécutés depuis la racine du projet :
```bash
python scripts/<nom_du_script>.py
```

Ou depuis le dossier scripts :
```bash
cd scripts
python <nom_du_script>.py
```
